Bachelor/Master Thesis: Trust and media manipulation I

Trust and media manipulation I. Bachelor/Master Thesis.

Social media are a key source of information for society, but it has been proven that can be also an effective source of misinformation if the readers are not critical or knowledgable enough. The goal of this project is to make a randomized controlled experiment to test if and how much sustainability experts can be manipulated.

The first part of the project consists on exploring what type of platforms we can use to run experiments like the one described below. Alternatively, if we cannot find a viable platform, you will build a small web prototype.

During the second part, you will study how we can manipulate sustainability experts by asking them to mark a collection of Twitter messages as true or false. The collection will contain original true or false statements that you will have sampled and curated beforehand, as well as some variations that you will generate. The objective of the variations is to make the statement look more or less credible than the original (e.g., we expect an statement to be more credible if we add a reference at the end, even if it false; while we expect statements with spelling errors to be perceived as less credible)

Finally, you will make a comparative analysis between each pair of original and manipulated statement to infer if there are particular aspects of the language that would make sustainability claims more or less credible.
