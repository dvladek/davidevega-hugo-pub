hugo-Uppsala Theme

hugo-Uppsala Theme. A beta version of an academic theme for Hugo powering the current website.

https://github.com/dvladek/hugo-uppsala

The hugo-Uppsala is an academic theme I developed over the years to build my personal webpage. This theme has been inspited by the Hugo Finisterre theme by Jonathan Gilligan and some other academics’ personal pages I have visited in the past.

The theme supports most of the features from Hugo Finisterre, such as the automatic generation of publication lists from BibTeX files and the dynamic generation of news in the homepage.

In contrast, this Hugo Finisterre it is a more lightweight theme redesigned from scratch using Bootstrap 5, Bootstrap Icons and SASS.
