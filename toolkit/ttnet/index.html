Temporal Text Network Library

Temporal Text Network Library. A C++ open source library for network modelling and analysis.

https://pypi.org/project/ttnet/

The ttnet (Temporal Text Network Library) is a python open source package for modelling and analizing communication networks, built on top of the uunet – the core library for network modelling and analysis developed and maintained by the Uppsala University Information Laboratory . ttnet is still in an early (but continuous) development.

The library is based on the Foundations of Temporal Text Networks and Metrics for Temporal Text Networks papers.
